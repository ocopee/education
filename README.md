# education

Development Courses - Submit Your Progress

A centralized repository for students to submit their practice exercises and exam solutions after completing coursework.

## Structure

Each student creates their own folder using their GitHub username and submits their work within that directory.

```
education/
├── kubernetes/
│   ├── student-username-1/
│   │   ├── practice-1/
│   │   ├── practice-2/
│   │   └── final-exam/
│   └── student-username-2/
│       ├── lab-1/
│       └── midterm-exam/
├── docker/
│   ├── student-username-1/
│   │   ├── containerization-lab/
│   │   └── docker-compose-project/
│   └── student-username-3/
│       ├── practice-1/
│       └── final-project/
├── nodejs/
│   ├── student-username-2/
│   │   ├── express-api/
│   │   ├── authentication-lab/
│   │   └── final-project/
│   └── student-username-4/
│       ├── rest-api-practice/
│       └── exam-solutions/
└── README.md
```
